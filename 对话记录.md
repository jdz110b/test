## 使用方法

1. **启动服务**
```bash
cd /Users/zjt/workspace/llmCreator
python3 app.py
```

2. **访问平台**
打开浏览器访问 `http://localhost:5000`

3. **配置大模型**
- 进入"配置管理"页面
- 添加大模型配置（API URL、API Key、模型名称）
- 测试连接

4. **上传语料**
- 选择语料类型（Question / QA 对）
- 上传 CSV 或 TXT 文件
- 系统自动解析

5. **执行评测**
- 选择要执行的评测操作
- 选择大模型配置
- 点击"执行选中操作"
- 查看评测结果

6. **导出结果**
- 点击"导出结果"按钮
- 下载 Excel 文件

## 技术栈

- **后端**: Python 3.10+, Flask, Flask-SQLAlchemy
- **前端**: HTML5, CSS3, JavaScript, Bootstrap 5
- **数据库**: SQLite
- **大模型**: OpenAI 兼容 API（支持阿里云通义千问等）
- **文件处理**: pandas, chardet, openpyxl

## 注意事项

1. **大模型配置**: 使用前必须先配置大模型 API
2. **文件编码**: 平台自动检测编码，支持 UTF-8、GBK 等
3. **API 调用**: 评测过程会调用大模型 API，请注意 API 调用次数和费用
4. **数据安全**: 所有数据存储在本地 SQLite 数据库中
5. **浏览器兼容**: 建议使用 Chrome、Firefox、Edge 等现代浏览器
