# 五子棋游戏

一个使用 Python 和 Tkinter 实现的五子棋游戏。

## 功能特点

- 双人对战模式
- 15x15 标准棋盘
- 图形化界面
- 自动判断胜负
- 支持重新开始

## 运行要求

- Python 3.x
- Tkinter（通常随 Python 一起安装）

## 运行方法

```bash
python gomoku.py
```

## 游戏规则

1. 黑棋先行，双方轮流下棋
2. 在棋盘交叉点上落子
3. 先连成五子（横、竖、斜）的一方获胜
4. 点击"重新开始"按钮可以重新开始游戏

## 游戏界面

- 顶部显示当前玩家
- 中间是棋盘
- 底部有"重新开始"和"退出"按钮

## 技术实现

- 使用 Tkinter Canvas 绘制棋盘和棋子
- 使用二维数组存储棋盘状态
- 实现了胜负判断算法（检查四个方向）
